## Description
Open-source anonymous civic participation platform for Greek citizens.
Vote on real Hellenic Parliament bills using cryptographic identity verification.
No account required. No data stored. Ed25519 cryptography.

## Anti-Features
- NonFreeNet: HLR Lookup service + Expo Push Notifications (FCM)

## Why IzzyOnDroid?
F-Droid MR pending (Draft) but React Native/Expo build chain requires custom setup.
IzzyOnDroid pre-built APK distribution is the right first step.
