Setup

# Clone git clone https://github.com/NeaBouli/pnyx && cd pnyx # API cd apps/api && python3 -m venv .venv && source .venv/bin/activate pip install -r requirements.txt # Web cd ../web && npm install # Docker cd ../../infra/docker && docker compose up -d

Tests

cd apps/api && pytest tests/ -v # 44 tests cd packages/crypto && pytest tests/ -v # 12 tests cd apps/web && npm run build # type check

Commit Format

feat(module): new feature fix(module): bug fix docs: documentation chore: infrastructure test: tests
