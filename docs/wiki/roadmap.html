Πώς λειτουργεί η εκκλησία

Όλα όσα πρέπει να ξέρετε — με απλά λόγια.

← Αρχική

🏛️ Τι είναι η εκκλησία;

Η εκκλησία είναι μια ψηφιακή πλατφόρμα που σας επιτρέπει να ψηφίζετε σε πραγματικά νομοσχέδια της Βουλής — χωρίς κόμμα, χωρίς μεσάζοντα, χωρίς γραφειοκρατία. Ψηφίζετε ανώνυμα από το κινητό σας, όποτε θέλετε. Τα αποτελέσματα δημοσιεύονται ανοιχτά ώστε να φαίνεται τι θέλουν πραγματικά οι πολίτες.

Δεν χρειάζεται λογαριασμό, email ή κωδικό. Μόνο το κινητό σας τηλέφωνο.

📱 Φάση Β — Τώρα

Η πλατφόρμα βρίσκεται σε φάση Beta. Σήμερα λειτουργεί ως εξής:

  1. Δίνετε τον αριθμό κινητού σας. Λαμβάνετε ένα SMS με κωδικό.
  2. Πληκτρολογείτε τον κωδικό. Το σύστημα επιβεβαιώνει ότι ο αριθμός σας είναι πραγματικός (HLR έλεγχος).
  3. Δημιουργείται αυτόματα ένα μοναδικό κρυπτογραφικό κλειδί στη συσκευή σας. Αυτό είναι η «ταυτότητά» σας στην πλατφόρμα.
  4. Ψηφίζετε! Κάθε ψήφος σας έχει βαρύτητα 1.0x.

Ο αριθμός τηλεφώνου σας διαγράφεται αμέσως μετά την επαλήθευση. Δεν αποθηκεύεται πουθενά.

Beta — Ενεργό

🔑 Πώς δημιουργείται και φυλάσσεται το κλειδί σας

Σκεφτείτε το κλειδί σαν μια ψηφιακή υπογραφή — σαν το μοναδικό γράψιμο του χεριού σας, αλλά στον ψηφιακό κόσμο.

Τι συμβαίνει βήμα-βήμα:

  1. Όταν επαληθεύεστε, η συσκευή σας (κινητό ή υπολογιστής) δημιουργεί ένα ζευγάρι κλειδιών: ένα ιδιωτικό και ένα δημόσιο.
  2. Το ιδιωτικό κλειδί μένει ΜΟΝΟ στη συσκευή σας. Κανείς άλλος δεν το βλέπει — ούτε εμείς.
  3. Το δημόσιο κλειδί στέλνεται στον server για να μπορεί να επαληθεύσει ότι μια ψήφος ήρθε πράγματι από εσάς.
  4. Όταν ψηφίζετε, η συσκευή σας «υπογράφει» την ψήφο με το ιδιωτικό κλειδί. Κανείς δεν μπορεί να πλαστογραφήσει την ψήφο σας.

Αν χάσετε τη συσκευή σας, χάνετε και το κλειδί. Μπορείτε να ξανα-επαληθευτείτε με νέο SMS και να πάρετε νέο κλειδί.

🏗️ Φάση Α — Έρχεται

Στην επόμενη φάση, θα μπορείτε να συνδεθείτε μέσω gov.gr — την πλατφόρμα του ελληνικού Δημοσίου.

  • Σύνδεση μέσω gov.gr OAuth (ίδιος κωδικός που χρησιμοποιείτε για taxisnet, e-ΕΦΚΑ κ.λπ.).
  • Οι Δήμοι ενεργοποιούν τοπικές ψηφοφορίες για τη γειτονιά σας.
  • Η ψήφος σας αποκτά βαρύτητα 2.0x — διπλάσια ισχύ.

Η Φάση Α ξεκινάει μόλις: 500+ ενεργοί χρήστες + 3+ συνεργασίες με ΜΚΟ + δημοσιευμένο roadmap.

Alpha — Σχεδιασμός

⚖️ Πώς υπολογίζεται η ψήφος σας

Κάθε ψήφος μετράει, αλλά η βαρύτητα εξαρτάται από τον τρόπο επαλήθευσής σας:

Μέθοδος Βαρύτητα Κατάσταση
SMS Επαλήθευση 1.0x Ενεργό
SMS + Πολιτική Πυξίδα 1.2x Ενεργό
gov.gr OAuth 2.0x Φάση Α

Η βαρύτητα 1.2x δίνεται αυτόματα αν έχετε συμπληρώσει την Πολιτική Πυξίδα. Δεν χρειάζεται κάποια άλλη ενέργεια.

🧭 Η Πολιτική Σας Πυξίδα

Η Πολιτική Πυξίδα είναι ένα ερωτηματολόγιο 48 ερωτήσεων που σας βοηθάει να δείτε πού στέκεστε πολιτικά — χωρίς ετικέτες.

  • Θέματα: υγεία, παιδεία, οικονομία, άμυνα, μεταναστευτικό, περιβάλλον κ.ά.
  • Τα δεδομένα μένουν 100% στη συσκευή σας. Δεν στέλνονται πουθενά — ούτε στον server μας.
  • Είναι κρυπτογραφημένα τοπικά (AES-256) — ακόμα κι αν κάποιος πάρει τη συσκευή σας, δεν μπορεί να τα διαβάσει.
  • Ο πυξίδα ενημερώνεται αυτόματα κάθε φορά που ψηφίζετε — γίνεται πιο ακριβής με τον καιρό.

Είναι εντελώς προαιρετικός. Μπορείτε να ψηφίζετε κανονικά χωρίς αυτόν.

🔔 Ειδοποιήσεις

Θα μπορείτε να λαμβάνετε ειδοποιήσεις όταν υπάρχει νέα ψηφοφορία — αλλά μόνο αν εσείς το θέλετε.

  • Εσείς επιλέγετε τις κατηγορίες: Βουλή, Δήμος, ή και τα δύο.
  • Ο server στέλνει μόνο ένα σύντομο σήμα (ping) — χωρίς περιεχόμενο. Η συσκευή σας κατεβάζει μετά τις λεπτομέρειες.
  • Μπορείτε να επιλέξετε τον ήχο ειδοποίησης ξεχωριστά για κάθε κατηγορία.
  • Μπορείτε να τις απενεργοποιήσετε ανά πάσα στιγμή.

Ιδιωτικότητα: ο server δεν ξέρει ποιος λαμβάνει τι. Η ειδοποίηση είναι ανώνυμη.

Σχεδιασμός

🏢 gov.gr — Ο Ρόλος των Δήμων

Στη Φάση Α, οι Δήμοι της Ελλάδας θα μπορούν να χρησιμοποιήσουν την εκκλησία για να ρωτήσουν τους κατοίκους τους σε τοπικά θέματα.

  • Π.χ. «Να γίνει πεζόδρομος η οδός Ερμού;» — ψηφίζουν μόνο οι κάτοικοι του Δήμου.
  • Η ταυτοποίηση γίνεται μέσω gov.gr — ο Δήμος δεν βλέπει ποιος ψήφισε τι.
  • Τα αποτελέσματα δημοσιεύονται ανοιχτά.

Είστε Δήμαρχος ή εργάζεστε σε Δήμο; Μάθετε πώς να ενεργοποιήσετε τον Δήμο σας →

🗳️ Αρχιτεκτονική Ψηφοφορίας

Πώς λειτουργεί η ψηφοφορία τεχνικά — με απλά λόγια:

Εσείς ψηφίζετε → Η συσκευή υπογράφει → Ο server επαληθεύει → Αρχειοθέτηση στο Arweave
  1. Επιλέγετε ΝΑΙ ή ΟΧΙ σε ένα νομοσχέδιο.
  2. Η συσκευή σας υπογράφει την ψήφο με το ιδιωτικό κλειδί (Ed25519). Ο server δεν ξέρει ποιος είστε — ξέρει μόνο ότι η ψήφος είναι αυθεντική.
  3. Ο server ελέγχει ότι δεν έχετε ξαναψηφίσει στο ίδιο νομοσχέδιο.
  4. Η ψήφος αρχειοθετείται μόνιμα στο Arweave — ένα αμετάβλητο αρχείο που κανείς δεν μπορεί να σβήσει ή να αλλάξει.

Κανείς — ούτε η πλατφόρμα, ούτε ο Δήμος, ούτε η κυβέρνηση — δεν μπορεί να δει ποιος ψήφισε τι. Η ανωνυμία δεν είναι επιλογή, είναι εγγύηση.
