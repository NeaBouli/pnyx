Community Driven

Πώς λειτουργεί η εκκλησία χωρίς ομάδα διαχείρισης

← Αρχική

Καμία εταιρεία. Καμία ομάδα.

Η εκκλησία δεν ανήκει σε κανέναν. Ο κώδικας είναι ανοιχτός (MIT License), τα δεδομένα είναι δημόσια (CC BY 4.0), και η πλατφόρμα χρηματοδοτείται αποκλειστικά από πολίτες.

Κόστος: ~93€/έτος (Server + Domain). Ανάπτυξη: 100% εθελοντική. Κανένα κέρδος.

Πώς λειτουργεί το Ticket System (POLIS)

Πολίτης → GitHub OAuth → Δημιουργία Ticket → GitHub Issues → Community Vote

Backend: GitHub Issues

Κάθε ticket αποθηκεύεται ως GitHub Issue. Κανείς δεν μπορεί να τα σβήσει.

Auth: Cloudflare Worker

OAuth token exchange μέσω Cloudflare — κανένα password στον server.

Ψήφος: Reactions

Κάθε ticket ψηφίζεται μέσω Reactions. Δημόσια, ανοιχτά.

Moderation: Κανένας

Δεν υπάρχει moderator. Η community αυτορρυθμίζεται.

Πώς να συμμετέχεις

Αναφέρετε σφάλμα ή πρόταση

  1. Community Board
  2. Κάντε κλικ «Νέο Ticket»
  3. Συνδεθείτε με GitHub (OAuth)
  4. Περιγράψτε το ζήτημα
  5. Η community ψηφίζει

Ψηφίστε σε υπάρχοντα tickets

Κάθε ticket έχει reactions. Κάντε κλικ στο αντίστοιχο emoji για να ψηφίσετε.

Συνεισφέρετε κώδικα

Fork → Branch → Pull Request. Οδηγός Contributing

Χρηματοδότηση

Η εκκλησία χρηματοδοτείται μόνο από πολίτες. Τα έξοδα είναι δημόσια στο community.html.

  • Server: 7€/μήνα (Hetzner)
  • Domain: 9,30€/έτος (Papaki)
  • Arweave: ~$0.0001/Upload (Community Wallet)
  • HLR: ~$0.002/Query (Community Wallet)
  • Ανάπτυξη: 100% εθελοντική
