Ταυτότητα — gov.gr OAuth

Αυθεντικοποίηση μέσω gov.gr με χρήση ΑΦΜ. Μόνο εξουσιοδοτημένοι αιρετοί και δημόσιοι υπάλληλοι αποκτούν πρόσβαση.

🗺️

Ψήφος κατά Εκλογική Περιφέρεια

Όλα τα δεδομένα είναι δημόσια. Η ψήφος περιορίζεται στον δήμο/περιφέρειά σας. Εθνικά: ψηφίζουν όλοι. Τοπικά: μόνο κάτοικοι.

🔐

2FA — Διπλή Επαλήθευση

Google Authenticator (TOTP) ή SMS. Email αποκλείεται ρητά ως 2FA μέθοδος λόγω ευπάθειας. Session: 8 ώρες.

Διαδικασία Ενεργοποίησης

1

Υποβολή Αίτησης

Αυτόματο

Ο Δήμαρχος ή εξουσιοδοτημένος εκπρόσωπος υποβάλλει αίτηση μέσω gov.gr OAuth. Το ΑΦΜ επαληθεύεται αυτόματα.

2

Επιλογή Υποδομής

Wizard

Ο Δήμος επιλέγει μεταξύ δύο μοντέλων λειτουργίας:

🏛️
Δικός σας Server

Υποχρεωτικό για δήμους >5.000 κατοίκων. Δικός σας κόμβος. Πλήρης έλεγχος.

☁️
Server ekklesia.gr

Για μικρούς δήμους (<5.000). Χωρίς κόστος server.

3

Επαλήθευση μέσω ΔΙ.ΑΥΓΕΙΑ

Αυτόματο

Το σύστημα επαληθεύει αυτόματα το αξίωμα μέσω ΔΙ.ΑΥΓΕΙΑ (diavgeia.gov.gr). Κάθε διορισμός δημάρχου δημοσιεύεται υποχρεωτικά με ΑΔΑ — η εκκλησία αντιπαραβάλλει αυτόματα το ΑΦΜ με τη δημόσια πράξη διορισμού. Κανένα χειροκίνητο βήμα.

4

Ενεργοποίηση Dashboard

Αυτόματο

Μοναδικός κωδικός ενεργοποίησης + gov.gr OAuth + ρύθμιση 2FA. Πλήρης πρόσβαση στα δεδομένα του δήμου σας.

5

Πρόσκληση Συνεργατών

Αυτόματο

Ο Δήμαρχος προσκαλεί συνεργάτες μέσω υπογεγραμμένου συνδέσμου (48h ισχύς). Κάθε συνεργάτης: gov.gr OAuth + 2FA. Δικαιώματα: μόνο ανάγνωση / εισαγωγή δεδομένων / πλήρης πρόσβαση εντός δήμου.

Ο κόμβος σας είναι έτοιμος να λειτουργήσει

Ένας κόμβος ekklesia.gr για τον Δήμο σας — αυτόνομος, ασφαλής, και πλήρως ελεγχόμενος από εσάς. Εγκατάσταση σε λιγότερο από 10 λεπτά.

$ curl -fsSL https://ekklesia.gr/node-setup.sh \
    | bash

Αυτό είναι το μόνο που χρειάζεται ο διαχειριστής του server σας.

🏛️

Πλήρης Αυτονομία

Ο κόμβος σας τρέχει στον δικό σας server. Τα δεδομένα των πολιτών σας παραμένουν αποκλειστικά στον Δήμο σας. Καμία εξάρτηση από τρίτους.

🔐

Στρατιωτική Ασφάλεια

Ed25519 κρυπτογράφηση — το ίδιο πρότυπο που χρησιμοποιεί το SSH και το Signal. Κάθε ψήφος υπογράφεται ψηφιακά. Αδύνατη η παραποίηση.

👁️

Ανοιχτός Κώδικας

100% ανοιχτός κώδικας στο GitHub (MIT). Ανεξάρτητος έλεγχος ασφαλείας ανά πάσα στιγμή. Κανένα κρυφό backdoor.

Τεχνικές Απαιτήσεις

Server Ubuntu 22.04+ ή Debian 12+
RAM 1GB minimum (2GB συνιστάται)
Αποθήκευση 10GB ελεύθερο χώρο
Κόστος ~10€/μήνα (VPS)

Το ekklesia.gr παρέχει δωρεάν: την subdomain [δήμος].ekklesia.gr, τεχνική υποστήριξη εγκατάστασης, και αυτόματες ενημερώσεις ασφαλείας.

Βήματα μετά την εγκατάσταση

1

DNS Ρύθμιση

Ορίστε DNS A-Record: [subdomain] → [IP του server σας] (5 λεπτά)

2

Ειδοποίηση ekklesia.gr

Στείλτε email: gov@ekklesia.gr — Ενεργοποίηση εντός 48 ωρών

3

gov.gr OAuth Αίτηση

Υποβάλετε αίτηση OAuth για το domain σας στο gov.gr
