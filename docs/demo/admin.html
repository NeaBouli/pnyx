Τεχνικά

Stack

Python FastAPI + PostgreSQL + Redis + Next.js + React Native (Expo) + Docker + Traefik

Deployment

Ένας Docker Compose stack. Τρέχει σε οποιοδήποτε VPS (Hetzner, DigitalOcean, AWS).

Κόστος

Server: ~10€/μήνα (CX23). SMTP: Brevo Free (300 emails/day). Arweave: ~0.001€/TX.
