Λειτουργίες Node Admin

Bill Lifecycle

Αυτόματη μετάβαση: ANNOUNCED → ACTIVE (14 ημέρες πριν) → WINDOW_24H (24ω πριν) → PARLIAMENT_VOTED → OPEN_END

ℹ️ Ο Lifecycle τρέχει κάθε 1 ώρα αυτόματα. Δεν απαιτεί χειροκίνητη παρέμβαση.

Arweave Αρχειοθέτηση

Κάθε ψηφοφορία που φτάνει σε PARLIAMENT_VOTED αρχειοθετείται αυτόματα στο Arweave blockchain.

Wallet: 2hkK3B...GdhBTs · Balance: 5.70 AR

ℹ️ Catch-up μηχανισμός: αν η αρχειοθέτηση αποτύχει, επαναλαμβάνεται κάθε 1 ώρα.

Telegram Bot

@ekklesia_news_bot — αυτόματες ειδοποιήσεις σε κανάλι + ομάδα. 11 θέματα (topics).

ℹ️ Κάθε αλλαγή κατάστασης bill → αυτόματο post στο σωστό topic.

Newsletter

Αυτόματο μηνιαίο newsletter μέσω Brevo API. Περιλαμβάνει: νέα bills, αποτελέσματα, Arweave αρχεία.

Monitor

6 κανόνες παρακολούθησης (Scraper, Lifecycle, Forum, Arweave, HLR). Telegram alerts αν κάτι πάει στραβά.

Embed Generator

Δημιουργία κώδικα ενσωμάτωσης (iframe) για εξωτερικές ιστοσελίδες. Vote + Results widgets.
