Αρχιτεκτονική Node

Multi-Tenant Αρχιτεκτονική: Κάθε Δήμος χρησιμοποιεί την ίδια υποδομή (API, DB) με φιλτράρισμα βάσει governance_level=MUNICIPAL + dimos_id. Τα δεδομένα είναι πλήρως απομονωμένα — κάθε Δήμος βλέπει μόνο τις δικές του ψηφοφορίες.
Demo περιορισμός: Αυτή η test node χρησιμοποιεί κοινό stack με το ekklesia.gr για λόγους πόρων. Σε παραγωγικό περιβάλλον, κάθε Δήμος μπορεί να έχει δικό του απομονωμένο stack (Docker Compose) για μέγιστη ασφάλεια.

Επίπεδα Απομόνωσης

Επίπεδο 1 — Multi-Tenant (Demo)

Κοινή DB + API. Κάθε Δήμος φιλτράρεται βάσει dimos_id. Row-Level Security. Κόστος: 0€ επιπλέον.

Επίπεδο 2 — Ξεχωριστό Stack (Παραγωγή)

Κάθε Δήμος: δικό API + DB + Web + Redis σε Docker Compose. Πλήρης απομόνωση. Κόστος: ~10€/μήνα VPS.

Επίπεδο 3 — Federated Network (Μέλλον)

Αυτόνομοι κόμβοι με Federation API. Κάθε Δήμος ανεξάρτητος, δεδομένα συγχρονίζονται. TrueRepublic bridge.

Δυνατότητες Admin

Δημιουργία Ψηφοφορίας

Τίτλος, περιγραφή, κατηγορία, ημερομηνία λήξης. Αυτόματο Forum topic + Telegram ειδοποίηση.

Lifecycle Αυτόματο

ANNOUNCED → ACTIVE → WINDOW_24H → PARLIAMENT_VOTED → OPEN_END. Χωρίς χειροκίνητη παρέμβαση.

Αποτελέσματα & Analytics

Real-time αποτελέσματα, CPLM, Divergence Score, Demographics (k-anonymity ≥10).

Embed Generator

Ενσωμάτωση ψηφοφοριών σε ιστοσελίδα του Δήμου μέσω iframe.

Newsletter + Telegram

Αυτόματο μηνιαίο newsletter (Brevo) + Telegram Bot ειδοποιήσεις.

Arweave Αρχειοθέτηση

Μόνιμη αρχειοθέτηση αποτελεσμάτων στο blockchain. Αδύνατο να αλλοιωθεί.

Τεχνικό Stack

Backend

Python FastAPI + PostgreSQL + Redis + Arweave + Ollama (AI)

Frontend

Next.js 14 (Web) + React Native / Expo (Mobile) + Discourse (Forum)

Infra

Docker Compose + Traefik (reverse proxy) + Let's Encrypt (SSL)

Κόστος Παραγωγής

Server: ~10€/μήνα (Hetzner CX23). SMTP: Brevo Free (300/ημέρα). Arweave: ~0.001€/TX.
