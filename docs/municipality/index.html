Για Αντιπροσώπους & Δήμους

Η εντολή ανήκει στον λαό

Εργαλεία διαφάνειας και λογοδοσίας για εκλεγμένους αντιπροσώπους — από τοπική αυτοδιοίκηση έως βουλή.

Τι προσφέρει η πλατφόρμα

Για κάθε εκλεγμένο αντιπρόσωπο — Βουλευτή, Περιφερειάρχη, Δήμαρχο ή Δημοτικό Σύμβουλο.

📱

Εφαρμογή εκπρόσωπος

Αποκλειστική εφαρμογή για εκλεγμένους αντιπροσώπους. Δείτε τα νομοσχέδια της περιοχής σας, αξιολογήσεις πολιτών και δείκτη απόκλισης.

📊

Δείκτης Απόκλισης

Αυτόματη σύγκριση μεταξύ της ψήφου σας στη Βουλή και της βούλησης των πολιτών. Διαφανής, δημόσιος, αμετάβλητος.

⛓️

Arweave Αρχειοθέτηση

Κάθε αποτέλεσμα ψηφοφορίας αρχειοθετείται μόνιμα στο blockchain Arweave. Αδύνατο να τροποποιηθεί ή να διαγραφεί.

🗳️

Ανώνυμη Συμμετοχή

Κρυπτογραφία Ed25519 — κανείς δεν γνωρίζει τι ψήφισε ο κάθε πολίτης, αλλά όλοι μπορούν να επαληθεύσουν το σύνολο.

📋

Δήλωση Αυτοδέσμευσης

Δημόσιο έγγραφο δέσμευσης για υποψήφιους αντιπροσώπους — συγκεκριμένες, μετρήσιμες υποχρεώσεις απέναντι στους πολίτες.

Πριν ακόμα ανακοινωθεί ένα ψηφοδέλτιο, ο υποψήφιος αντιπρόσωπος καλείται να υπογράψει δημόσια δεσμεύσεις: διαφάνεια, σεβασμός στη βούληση των πολιτών, λογοδοσία, και αποδοχή αξιολόγησης. Όχι ευχολόγια — συγκεκριμένες υποχρεώσεις.

Ενεργοποίηση Δήμου

Πώς ο Δήμος σας μπορεί να συμμετέχει επίσημα στην πλατφόρμα ekklesia.gr.

Επικοινωνία με την Πρωτοβουλία

Ο Δήμαρχος ή εξουσιοδοτημένος εκπρόσωπος επικοινωνεί μέσω της επίσημης φόρμας.

Μελλοντική επίσημη επαλήθευση

Σχεδιασμός Alpha για gov.gr OAuth ή νέο έγγραφο QR/eSeal. Δεν είναι ενεργό στη Beta και απαιτεί επίσημη έγκριση και security/privacy review.

Ενεργοποίηση αντιπροσώπου

Ο Δήμαρχος λαμβάνει invite code για την εφαρμογή εκπρόσωπος και ξεκινά να παρακολουθεί τις ψηφοφορίες.

Δημόσια Δήλωση Αυτοδέσμευσης

Προαιρετική αλλά ισχυρά συνιστώμενη υπογραφή της Δήλωσης AG-EKK-006 ενώπιον των πολιτών.
