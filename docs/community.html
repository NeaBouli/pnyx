Το POLIS Ticket System

Το POLIS είναι το κρυπτογραφικό σύστημα ανατροφοδότησης της εκκλησίας. Κάθε πολίτης δημιουργεί tickets μέσω ανώνυμης ταυτότητας Ed25519. Ταυτοποίηση με δημόσιο κλειδί (pk_polis) — χωρίς λογαριασμό, email ή κωδικό. Ο nullifier αποτρέπει διπλά tickets. 3 θετικές ψήφοι = η πρόταση προωθείται.

Για developers: Εναλλακτικά, αναφορές μέσω GitHub Issues χωρίς ταυτοποίηση.
